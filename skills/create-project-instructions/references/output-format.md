# Output Format: create-project-instructions

Return one complete markdown artifact and a short completion summary.

## Artifact contract
- Provide one markdown code block containing full project-context `AGENTS.md` output.
- Include explicit target path: `AGENTS.md` (or `<scope>/AGENTS.md` for nested-scope variants).

## Completion summary contract
Include:
- role count
- default session state highlights
- testing command coverage
- assumptions and unresolved gaps
